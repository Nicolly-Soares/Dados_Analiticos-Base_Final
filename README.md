Este projeto consiste na construção de um dashboard interativo para análise de dados da saúde suplementar no Brasil, 
com foco na evolução do número de beneficiários e nos custos médico-hospitalares (VCMH) entre os anos de 2018 a 2023.

O objetivo do projeto é transformar dados brutos em informações visuais claras e úteis, permitindo:

- Analisar o crescimento do número de beneficiários
- Avaliar a variação dos custos médico-hospitalares (VCMH)
- Identificar padrões e tendências ao longo do tempo
- Apoiar a tomada de decisão com base em dados

O dashboard tem como funcionalidade: 

- filtro por ano(selctbox), 
- cards interrativos, 
- graficos de linha, 
- insihts 
- uma conclusão analitica referente aos dados vissualizados.

- oscilaçao relevante no VCMH

- Demanda crescente demanda custos
- Queda no VCMH em 2020
- Tendência de alta ao longo prazo

conclusão

Os dados analisados indicam que o crescimento da saúde suplementar no Brasil vem acompanhado por um aumento nos custos médico-hospitalares, o que reforça a importância do monitoramento contínuo desses indicadores para planejamento estratégico e sustentabilidade do setor.
